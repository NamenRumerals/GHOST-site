## Decisions Made

### Tech Stack
- Plain HTML/CSS/JS, no framework, no build step
- Three.js via CDN for solar system
- GitHub API for gallery (reads folder contents dynamically — just drop images in folder and push)
- All content JSON-driven for non-technical staff

### Pages (12 total)
`index.html` + 7 division pages + `lore.html` + `events.html` + `staff.html` + `gallery.html`

### Navigation
`HOME · DIVISIONS ▾ · LORE · EVENTS · STAFF · GALLERY` + `[JOIN US]` button (Discord invite)

### Home Page Sections
1. Boot sequence (terminal + jitter animation + unicode loading bars → flicker into solar system)
2. Solar system hero (Three.js, full viewport)
3. About G.H.O.S.T.
4. Latest Events (3 cards)
5. Stats + Socials (TikTok links for main server + all 7 divisions)
6. Footer

### Boot Sequence Animation
- Random jitter timing: `delay = baseDelay + Math.random() * jitter` per message
- Unicode block bars: `░░░░░░░░░░` filling to `██████████`
- Skip button (click anywhere)
- Configurable constants in JS

### Solar System
- G.H.O.S.T. logo center, 7 divisions orbit on 2 rings (4 inner, 3 outer)
- Each orbit glows in division color
- Click division → popup (logo, name, motto, one-liner, "View Division" button)

### Staff Page
1. Owner — featured solo
2. Server Staff — Admins → Mods → Staff
- Division Commanders are ONLY on their division pages, not on Staff page

### Events Page
- Cards (optional art, name, short desc) sorted newest first
- Click → popup (name, date, division tags, full writeup, image gallery)
- No dedicated page per event

### Gallery
- Tabs: Media · Medals · Posters
- GitHub API reads folders dynamically
- Medals = earnable awards

### Division Pages
- Logo, name, motto, description, leadership, TikTok, media gallery
- No ops log (separate site handles that)

### Lore Page
- Origin story + timeline (JSON-driven, lore admin updates)

### Typography
- Wargate: image assets only (desktop license — cannot web embed)
- Orbitron: all live web text
- Share Tech Mono: body/terminal text

### Font License Note
Wargate desktop license purchased. Cannot use as web font. Export from design software as PNG/SVG for hero graphics only.
